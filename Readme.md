# 📂 Automated File Organizer

A simple desktop application built with **Python** and **Tkinter** that automatically organizes files into categorized folders based on their file extensions.

## ✨ Features

* User-friendly graphical interface (GUI)
* Select any folder from your computer
* Automatically creates folders if they do not exist
* Organizes files into categories such as:

  * Images
  * Documents
  * Excel
  * PowerPoint
  * Python
  * Videos
  * Music
  * Archives
  * Executables
  * Others
* Displays the selected folder path
* Shows progress while organizing files
* Displays a success message after completion
* Reset and Exit buttons
* Ignores existing folders while organizing

## 🛠 Technologies Used

* Python 3
* Tkinter
* OS Module
* Shutil Module

## 📁 Supported File Types

| Category    | Extensions                           |
| ----------- | ------------------------------------ |
| Images      | .png, .jpg, .jpeg, .gif, .bmp, .webp |
| Documents   | .txt, .pdf, .doc, .docx              |
| Excel       | .xls, .xlsx, .csv                    |
| PowerPoint  | .ppt, .pptx                          |
| Python      | .py                                  |
| Videos      | .mp4, .mkv, .avi, .mov               |
| Music       | .mp3, .wav, .aac                     |
| Archives    | .zip, .rar, .7z                      |
| Executables | .exe, .msi                           |
| Others      | Any unsupported file type            |

## 🚀 How to Run the Project

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open the project folder in Visual Studio Code or any Python IDE.
4. Open a terminal inside the project folder.
5. Run the following command:

```
python main.py
```

6. Click **Select Folder**.
7. Choose the folder you want to organize.
8. Click **Start Organizing**.
9. The files will automatically be sorted into their respective folders.
